# BookVerse - E-Commerce & Support Platform

BookVerse is a modern, full-featured web application that provides a complete e-commerce experience for bookstores. It includes customer-facing storefronts, an administration dashboard for inventory and order management, and a dedicated Customer Support Representative (CSR) portal for handling support tickets and inquiries.

## Key Features
- **Customer Portal:** Browse catalogs, manage shopping carts, track orders, and interact with support.
- **Admin Dashboard:** Manage inventory, track store performance, and monitor active orders and customers.
- **CSR Portal:** Access knowledge bases, manage and reply to customer support tickets, and review order statuses.

## Getting Started

1. Run `npm install`
2. Run `npm run dev`
